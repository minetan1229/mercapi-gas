# 使い方

## セットアップ

1. Google Apps Script の新規プロジェクトを作成します。
2. `mercapi.gs` をプロジェクトにコピーします。
3. 必要に応じて `exmaple.gs` をコピーして実行例として使います。

## 基本的な使い方

### 初期化

```javascript
var mercapi = new Mercapi();
```

`userAgent` を指定したい場合は `new Mercapi({ userAgent: '...' })` を使います。

### 検索

```javascript
function runMercariSearch() {
  var m = new Mercapi();
  var results = m.search('shrapnel', {
    price_min: 1000,
    price_max: 5000,
    sort_by: SearchRequestData.SortBy.SORT_CREATED_TIME,
    sort_order: SearchRequestData.SortOrder.ORDER_DESC,
  });

  Logger.log('Found: ' + results.meta.num_found);
  if (results.items.length > 0) {
    var item = results.items[0];
    Logger.log(item.name + ' / ' + item.price);
    Logger.log(item.full_item().description);
  }
}
```

`results.meta` には `num_found` / `next_page_token` / `prev_page_token` が入ります。  
`results.items` は検索結果の `SearchResultItem` 配列です。

`SearchResultItem` からは以下が使えます。

- `full_item()`：商品詳細 (`Item`) を取得
- `seller()`：出品者プロフィール (`Profile`) を取得
- `real_price`：値下げ不可商品の場合は `null`

### ページング

```javascript
var first = m.search('shrapnel');
var next = first.next_page();
var prev = next.prev_page();
```

ページトークンがない場合は `IncorrectRequestError` が投げられます。

### 商品詳細

```javascript
var item = m.item('m1234567890');
if (item) {
  Logger.log(item.name);
}
```

存在しない場合は `null` です。

### ユーザープロフィール/出品一覧

```javascript
var profile = m.profile('12345678');
var items = m.items('12345678'); // items.items に配列が入る
```

### 検索オプション

`search(query, options)` で使える主なオプション:

- `categories` / `brands` / `sizes`
- `price_min` / `price_max`
- `item_conditions`
- `shipping_payer`
- `colors`
- `shipping_methods`
- `status`
- `sort_by` / `sort_order`
- `exclude`（除外キーワード）
- `page_token`（次/前ページ用）

値の候補は以下の定数を参照してください。

- `SearchRequestData.SortBy`
- `SearchRequestData.SortOrder`
- `SearchRequestData.Status`
- `SearchRequestData.ShippingMethod`

## スプレッドシートでの自動検索例

`exmaple.gs` には以下の例が含まれています。

- 検索条件をシートから読み込んで定期実行
- 新着アイテムの通知メール送信
- 実行間の差分管理用シートの作成

利用する場合は、スプレッドシートを作成して Apps Script に貼り付けたあと、
`setupMercapiSearchTrigger()` を実行してトリガーを作成してください。
